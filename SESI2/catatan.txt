tag <a>
merupakan tag anchor atau jangkar atau link
ada 4 buah tipe link : 
1> relatif = link bergantung pada penempatan folder
2> absolute = link berupa http:\\
3> aplikasi link = link berupa menuju aplikasi ch: mailto
4> session link = merupak link dari id href"#bab2"

tag <li>
merupakan tag untuk list
ada 2 buah tipe list :
1> <ul> unordered list merupakan list yang tidak berurutan akan memunculkan "-"
2> <ol> ordered list merupakan list yang berurutan akan memunculkan "1,2,3"

tag <dl>
merupakan tag untuk data list
<dl><dt><dd></dd></dt></dl>

tag <img>
merupakan tag untuk image
ada atribut khusus :
= src
= alt mempengaruhi SEO
= width
= height
tinypng dan tinyjpg
